# Complete HTML & CSS Portfolio

This repository contains a fully coded personal portfolio website built with **HTML** and **CSS**. The project is structured as a single-page application with smooth scrolling navigation and a clean, modern design.

##  Project Structure

##  Features

- **Responsive Navigation Bar** with smooth scroll links
- **Hero Section** with introduction and call-to-action
- **About Me** section with personal description
- **Skills** section with listed technologies
- **Projects** section with project cards
- **Contact** section with email and LinkedIn link
- **Modern CSS** with flexbox, custom colors, and smooth scrolling

##  Technologies Used

- HTML5
- CSS3
- Google Fonts (Arial)
- No external libraries or frameworks

##  Design Highlights

- Clean, minimal layout
- Sky blue background with gray text
- Accent color: `#f39c12` (orange/yellow)
- Rounded profile image
- Card-based project layout with left border accents
- LinkedIn button styled in brand color (`#0A66C2`)

##  Sections

1. **Home** – Hero introduction
2. **About** – Personal bio and goals
3. **Skills** – Programming languages and tools
4. **Projects** – Three showcased projects
5. **Contact** – Email and LinkedIn link

##  How to Use

1. Clone or download the repository.
2. Open `index.html` in your browser.
3. Update `me.jpeg` with your own profile picture (optional).
4. Edit the HTML and CSS files to personalize content and styling.

##  Customization

- Replace `"Shritij Jaiswal"` with your name
- Update `shritijjaiswal27@gmail.com` with your email
- Replace LinkedIn link with your own profile
- Modify colors in `style.css`
- Add or remove skills/projects as needed

## Notes

- The project uses **plain CSS** with no frameworks for simplicity and learning purposes.
- The navigation menu is fixed and responsive.
- The profile image path is `me.jpeg` – ensure the image exists in the same folder.
- Smooth scrolling is enabled via `scroll-behavior: smooth`.
